Projet créé par Sacha Pronost, dans le cadre d'une demande de stage pour l'entreprise Demand Side Instruments.
But de l'application : page internet permettant de rechercher des articles sur le WEB via l'api News API.

#LANCEMENT DE L'APPLICATION#

L'API News Api propose une clé d'api gratuite, mais qui ne peut être utilisé seulement dans un cadre de développement
interne, ainsi mon application (dans l'état actuel) ne peut pas être diffusé sur le web et doit être lancée seulement
dans votre localhost (il est possible de la faire fonctionner sur internet, mais pour cela il faut payer la clé d'API).

Ainsi pour lancer mon site web, il faut tout d'abord lancer un localhost dans le répertoire contenant mon index.html.
Pour cela il existe plusieurs moyens, en voici deux:
-Sous Linux :
	1) placez-vous dans le répertoire de mon projet contenant le index.html
	2) Lancer un terminal dans ce répertoire
	3) Dans le terminal écrivait la commande suivante: python -m SimpleHTTPServer
	4) Dans votre navigateur web écrivait dans l'url: localhost:8000
-Sous Windows :
	1) Il faut Node.js déjà installé sur la machine (https://nodejs.org/en/)
	2) Dans le CMD entrait la commande : npm install http-server -g
	3) Placer le CMD dans le répertoire de mon projet contenant le index.html
	4) Dans le CMD entrait la commande : http-server
	5) Dans votre navigateur web écrivait dans l'url : localhost:8080

Il existe bien sûr bien d'autre moyen de faire la même opération.
